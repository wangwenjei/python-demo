# Python 常用模块用法

os     文件和路径操作功能库
shutil      高级的 文件/文件夹 拷贝与压缩 处理库  https://www.jianshu.com/p/7846b6cbe4c8
pathlib  Py3.5后支持的 文件路径处理库
sys    系统和环境相关功能库
time        时间库
datetime    日期处理库
random      随机数库
json     json 序列化库   序列化只支持全语言
pickle   pickle 序列化库 序列化只支持Python
configparser   ini文件解析器  可以用于定义读取配置文件
hashlib        hash算法库
subprocess     用于执行系统命令
logging     日志功能
re          正则表达式
