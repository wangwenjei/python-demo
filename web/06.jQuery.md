### jQuery语法

```angular2html
# jQuery基本语法
jQuery(选择器).action()
秉持着jQuery的宗旨 jQuery简写    $
jQuery()  === $()

# jQuery与js代码对比
eg:将p标签内部的文本颜色改为红色
// 原生js代码操作标签
let pEle = document.getElementById('d1')
pEle.style.color = 'red'

// jQuery操作标签
$('p').css('color','blue')
```

## 查找标签
### 基本选择器
```angular2html
// id选择器
$('#d1')

// class选择器
$('.c1')

// 标签选择器
$('span')

"""一定要区分开(重点)"""

$('#d1')[0]
===>
<div id="d1">…</div>

document.getElementById('d1')
===>
<div id="d1">…</div>

// 标签对象如何转jQuery对象

```


### 组合选择器/分组与嵌套
```angular2html
$('div')
$('div.c1')  找div标签且,class包含c1
$('div#d1')  找div标签且,包含id名为d1
$('*')  找所有

$('#d1,.c1,p')  # 并列+混用

$('div span')  # 后代
$('div>span')  # 儿子
$('div+span')  # 毗邻
$('div~span')  # 弟弟

```

### 筛选器方法
```angular2html
$('#d1').next()  # 同级别下一个标签
   ===>  w.fn.init [span, prevObject: w.fn.init(1)]0: spanlength: 1prevObject: w.fn.init
$('#d1').nextAll() # 同级别下所有
$('#d1').nextUntil('.c1')  # 直到.c1这个样式为止,且不包括c1


$('.c1').prev()  # 同级别上一个标签
$('.c1').prevAll()
$('.c1').prevUntil('#d2')


$('#d3').parent()  # 第一级父标签
$('#d3').parent().parent()  # 父级标签的父级标签
...
$('#d3').parents()  # 拿到祖宗十八代的所有父级标签,一直向上找,直到定位到html标签
$('#d3').parentsUntil('body')  # 一直找父级标签直到遇到body标签


$('#d2').children()  # 儿子标签

$('#d2').siblings()  # 同级别上下所有


$('div p')
# 等价           
$('div').find('p')  # find从某个区域内筛选出想要的标签 

$('div span:first')
# 等价
$('div span').first()

$('div span:last')
# 等价
$('div span').last()

$('div span:not("#d3")')
# 等价
$('div span').not('#d3')

```
