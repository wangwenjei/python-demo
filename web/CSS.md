
### CSS
css(层叠样式表): 就是给HTML标签添加样式,让他变得更加好看

CSS语法结构
```angular2html
选择器 {
    属性1: 值1;
    属性2: 值2;
    属性3: 值3;
}
```

CSS三种引入方式
```angular2html
1. style标签内直接书写
    <style>
        h1 {
          color: burlywood;
        }
    </style>
2. link标签引入外部CSS文件(最正规的方式)
    <link rel="stylesheet" href="mycss.css">
3. 行内式(一般不用)
    <h1 style="color: green"> Hello CSS </h1>
```
